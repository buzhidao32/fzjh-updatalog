## 📝 更新类型说明

| 类型 | 说明 |
|------|------|
| 新增 | 新武学、新招式、新残页的上线 |
| 调整 | 武学数值、效果、冷却时间等的平衡性调整 |
| 重构 | 武学招式的重新设计，通常伴随动画和效果的全面更新 |
| 修复 | 武学相关的Bug修复 |
